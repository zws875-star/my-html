# 天气工具 PWA 版

## 文件
- index.html
- manifest.webmanifest
- service-worker.js
- icons/

## 部署到 GitHub Pages
1. 新建 GitHub 仓库
2. 上传以上所有文件和文件夹到仓库根目录
3. Settings -> Pages
4. 选择 Deploy from a branch
5. 选择 main 和 /root
6. 保存后等待部署

## iPhone 添加到主屏幕
1. 用 Safari 打开 GitHub Pages 链接
2. 点击底部分享按钮
3. 选择“添加到主屏幕”

## 说明
- 首次使用需要联网查询天气
- 查询结果会缓存在浏览器本地
- 这是基础 PWA 版，适合继续升级
